## 1. 实现概述

### 1.1 实现范围

本次实现完成了记忆归纳总结模块的核心功能（P0 需求）：

- 流水记忆层（StreamMemory）的存储与查询
- 长期记忆层（LongTermMemory）的存储与查询
- 标准接口：`write_memory()` 和 `search_memory()`
- 记忆升级机制（定时任务将流水升级为长期记忆）
- 事件监听与触发（对话完成事件处理）
- 初步总结与长期记忆提炼（使用 LLM）
- 模型分离配置（独立配置记忆归纳模型）

### 1.2 未实现功能（后续迭代）

- 向量化存储与检索（EmbeddingConfig 已预留）
- 指代消除功能
- 多租户支持
- 记忆可视化界面
- 记忆手动编辑

### 2.2 数据表结构

**stream_memories 表：**
```sql
CREATE TABLE stream_memories (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    trace_id TEXT NOT NULL,
    session_key TEXT,
    channel_type TEXT,
    content TEXT,
    summary TEXT,
    event_type TEXT,
    created_at DATETIME NOT NULL,
    processed INTEGER DEFAULT 0,
    processed_at DATETIME
);

CREATE INDEX idx_stream_trace_id ON stream_memories(trace_id);
CREATE INDEX idx_stream_session_key ON stream_memories(session_key);
CREATE INDEX idx_stream_created_at ON stream_memories(created_at);
CREATE INDEX idx_stream_processed ON stream_memories(processed);
CREATE INDEX idx_stream_channel ON stream_memories(channel_type);
```

**long_term_memories 表：**
```sql
CREATE TABLE long_term_memories (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    date TEXT NOT NULL,
    summary TEXT,
    what_happened TEXT,
    conclusion TEXT,
    value TEXT,
    highlights TEXT,
    source_ids TEXT,
    vector_id TEXT,
    created_at DATETIME NOT NULL,
    updated_at DATETIME NOT NULL
);

CREATE INDEX idx_longterm_date ON long_term_memories(date);
```

## 3. 核心流程实现

### 3.1 写入流程

```
对话完成事件
    │
    ▼
MemoryEventHandler.OnConversationCompleted()
    │
    ├───▶ Summarizer.SummarizeConversation() // 生成初步总结
    │
    ▼
MemoryService.WriteMemory()
    │
    ├───▶ StreamMemoryRepository.Create() // 写入 SQLite
    │
    ▼
返回成功（异步）
```

### 3.2 查询流程

```
调用 SearchMemory(query, filters)
    │
    ├───▶ StreamMemoryRepository.FindByTimeRange() // 近期流水记忆
    │
    ├───▶ LongTermMemoryRepository.SearchByKeyword() // 长期记忆
    │
    ▼
合并结果，按时间排序
    │
    ▼
返回 SearchResult
```

### 3.3 定时升级流程

```
定时任务触发（每日 05:30-06:30）
    │
    ▼
MemoryUpgradeJob.Run()
    │
    ├───▶ StreamMemoryRepository.FindByTimeRange() // 查询昨日流水
    │
    ├───▶ Summarizer.SummarizeToLongTerm() // 提炼精华
    │
    ├───▶ LongTermMemoryRepository.Create() // 创建长期记忆
    │
    ├───▶ StreamMemoryRepository.MarkAsProcessed() // 标记已处理
    │
    ▼
完成
```

## 5. 集成指南

### 5.1 初始化记忆模块

```go
// 1. 获取数据库连接
import "gorm.io/gorm"

// 2. 创建仓储层
streamRepo := repository.NewStreamMemoryRepository(db)
longTermRepo := repository.NewLongTermMemoryRepository(db)

// 3. 创建 Summarizer（需要实现 LLMClient 接口）
summarizer := service.NewMemorySummarizer(llmClient, "", "")

// 4. 创建 Service
memoryService := service.NewMemoryService(streamRepo, longTermRepo, summarizer, config.Memory.Enabled)

// 5. 创建事件处理器
eventHandler := handler.NewMemoryEventHandler(memoryService, nil, summarizer, logger, config.Memory.Enabled)

// 6. 注册到事件总线（示例）
eventBus.Subscribe("conversation.completed", eventHandler.OnConversationCompleted)

// 7. 注册定时任务
upgradeJob := job.NewMemoryUpgradeJob(memoryService, logger, config.Memory.Scheduled.Enabled, config.Memory.Scheduled.TimeWindow, config.Memory.Scheduled.Timezone)
cron.AddFunc("0 30 5 * * *", job.MemoryUpgradeJobFunc(upgradeJob))
```

### 5.2 使用记忆接口

```go
// 写入记忆
metadata := map[string]interface{}{
    "trace_id":     "trace-001",
    "session_key":  "matrix:room-id",
    "channel_type": "matrix",
}
err := memoryService.WriteMemory(ctx, "对话总结内容", metadata)

// 搜索记忆
filters := models.SearchFilters{
    Query: "项目讨论",
    Limit: 20,
}
result, err := memoryService.SearchMemory(ctx, "项目讨论", filters)
```

## 6. 后续优化方向

### 6.1 功能增强

1. **向量化检索**
   - 实现 EmbeddingClient 接口
   - 集成 ChromaDB 或 Milvus
   - 语义搜索替换关键词搜索

2. **指代消除**
   - 读取对话前后上下文
   - 解析代词指代关系
   - 提升总结质量

3. **记忆管理界面**
   - Web 界面查看记忆时间线
   - 支持搜索、过滤、导出
   - 支持手动编辑长期记忆

### 6.2 性能优化

1. **批量处理优化**
   - 优化大批量升级的性能
   - 支持增量升级

2. **查询优化**
   - 添加复合索引
   - 支持全文搜索

---

## 7. 已知限制

1. **向量化未实现**：当前仅有关键词搜索，语义搜索待后续迭代
2. **LLM 依赖**：总结质量依赖外部 LLM，成本较高
3. **单实例存储**：当前基于 SQLite，不支持分布式部署
4. **无上下文感知**：指代消除功能未实现
